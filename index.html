Wintergatan MidiPunch

Open source project om MIDI bestanden uit te snijden uit papier met een lasercutter om muziekdozen te bespelen.

Dit project is gestart als antwoord op een open vraag die Martin Molin van de groep Wintergatan (bekend van de Marble Machine) heeft gesteld in deze video. Na enkele mails met Martin hebben we een GitHub repository aangemaakt en heeft hij me project manager gemaakt. De software is nog in prototype fase, maar er wordt druk gewerkt met ontwikkelaars van over heel de wereld om het af te krijgen!

Het programma is geschreven als web app zonder back-end zodat iedereen het kan gebruiken, zelfs offline.

Eenakter

Foto: Luc Huygens

Eenakter is een festival waar drie scholen hun theaterstudenten de opdracht geeft om een theaterstuk zelf te schrijven en op te voeren. Het stuk MarCel werd geschreven door drie medestudenten die mij dan vriendelijk hebben gevraagd of ik licht en geluid wou verzorgen tijdens hun optredens. Als extraatje heb ik dan ook beide opvoeringen van alledrie de deelnemers gefilmd. Die opnames heb ik dan verwerkt tot één grote DVD.

Artikel op Nieuwsblad.be

Gebruikte technologieën:

  • Adobe Premiere
  • Adobe After Effects
  • Adobe Encore
